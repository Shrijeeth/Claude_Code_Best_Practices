Claude Code Architecture
Simplicity by Design
Claude Code's architecture is surprisingly simple and doesn't rely on complex infrastructure:
What Claude Code DOES Use:

✅ Pattern searching within code files
✅ Directory listing
✅ File viewing
✅ Regex operations
✅ Simple text-based tools

What Claude Code DOES NOT Use:

❌ Semantic embedding of code
❌ Vector databases
❌ Complex searchable structures
❌ Code indexing

The code.md File: Claude's Memory System
markdown# code.md - Claude's Note-Taking System

Claude autonomously:
1. Reads through your codebase
2. Takes notes in code.md
3. Figures out what's happening in the codebase
4. Drives decision-making based on these notes
5. Advances your code systematically
```

**Key Benefits:**
- Autonomous understanding of codebase structure
- Self-directed exploration and learning
- Informed decision-making
- No external indexing required
```
